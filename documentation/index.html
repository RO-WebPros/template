Installing the Template

After unzip the download pack, you'll find a Template Folder with all the files.

You can view this Template in any browser, you can display or edit the Template without an internet connection.

Now open your FTP Client (like Filezilla) or directly through cpanel and upload the content of the Template on your server root.

Once the files are done uploading go to www.yourdomainname.com/index.html

HTML Structure

This Template is based on the Bootstrap Framework. It uses the default Bootstrap grid system of 12 columns and all its other features. For more information on how to use it visit Bootstrap.

Basic Grid HTML

For a simple two column layout, create a .row and add the appropriate number of .col-md* columns.

Given this example, we have .col-md-4 and .col-md-8, making for 12 total columns and a complete row.


	<div class="row">  
	    <div class="col-md-4">...</div>  
	    <div class="col-md-8">...</div>  
	</div>

Nesting Columns

To nest your content with the default grid, add a new .container and set of .col-md* columns within an existing .col-md* column. Nested rows should include a set of columns that add up to the number of columns of its parent. 


	<div class="container">  
	    <div class="col-md-9">  
	        Level 1 column  
	        <div class="row">  
	            <div class="col-md-8">Level 2</div>  
	            <div class="col-md-4">Level 2</div>  
	        </div>  
	    </div>  
	  
	    <div class="col-md-3">  
	        Level 2 column  
	    </div>  
	</div>

General HTML Structure

The HTML structure is well organized. It has a wrapper that contains everything. Inside it you will find 2 main sections: a header containing the main content which can be seen on the actual page, and an "information" div containing the 3 pages with additional info.

Inside index.html:


  	<div class="wrapper">

  		<header class="main-content">
		...
  		</header>



		<div class="information">
		...
 		</div>

	</div>

Javascript Structure

These are the JS files that are loaded at the end of the Body Section.

	<!-- jQuery -->

	<!-- Bootstrap -->
	<script src="js/bootstrap.min.js"></script>

	<!-- Pace.js for preloader -->

Customisation

Lets start customising the template to your needs. Open index.html and follow the steps.

Since most of these steps are same for all pages, will only explain it once here.

Logo

To change the logo text showing while the page loads:


	<div class="loader">
		<h1 class="loader-text" data-content="CHANGE HERE">CHANGE HERE</h1>
	</div>

Navigation

The three navigation buttons have the following markup:


	<ul class="main-nav__links">

		<li class="main-nav__item">
			<a class="main-nav__link" data-toggle="modal" data-target="#about">
				<i class="glyphicon glyphicon-user"></i>
			</a>
			
		</li>

		<li class="main-nav__item">
			<a class="main-nav__link" data-toggle="modal" data-target="#services">
				<i class="glyphicon glyphicon-wrench"></i>
			</a>
		</li>

		<li class="main-nav__item">
			<a class="main-nav__link" data-toggle="modal" data-target="#contact">
				<i class="glyphicon glyphicon-envelope"></i>
			</a>
		</li>

	</ul>

Each of them is linked to the coresponding info section (which is a modal)
